# CryptoCoin
This is a cool cypto currency tracking app that fetches current rates of crypto currencies such as Bitcoin, Etherium and DodgeCoin.

# Technologies used
- Xcode
- Swift
- UIKit
- Public APIs
- Networking
- JSON Decoding
- JSON Parsing
- MVC Design Pattern
- Storyboards and programatic UI
- AutoLayout
- Segues and Navigation
